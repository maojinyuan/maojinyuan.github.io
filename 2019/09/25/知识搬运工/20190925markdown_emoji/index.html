基础篇

问题所在：Hexo默认采用的hexo-renderer-marked渲染器不支持插件拓展，因而解析不了emoji表情
解决办法：卸载原有渲染器，安装markdown-it渲染器

1.$ npm un hexo-renderer-marked --save //卸载原有marked渲染器
2.$ npm i hexo-renderer-markdown-it --save //安装markdown-it渲染器
3.$ npm install markdown-it-emoji --save //安装 markdown-it-emoji 插件
